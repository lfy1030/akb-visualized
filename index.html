AKB Visualized

This project aims to programmatically interpret and visualize selected music from popular Japanese girl group AKB48. The music is transcribed and then processed as data into several renditions of different visualizations. Check out the different data representation above! The source code is available on GitHub.

Disclaimer: I do not own any of AKB48's music. However, I did transcribe the songs and the transcription data can be found on GitHub as well.
